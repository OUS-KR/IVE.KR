English Suffix

‑ive

/ ɪv /

Definition

의미
"~하는 성질이 있는", "~하는 경향이 있는" — 동사·명사에 붙어 형용사를 형성
기능
형용사 형성 접미사 (일부 단어는 명사로도 사용: detective, executive 등)
어원
라틴어 ‑īvus → 앵글로노르만어 ‑if → 중세 영어 ‑yf → 현대 영어 ‑ive

Etymology

접미사 ‑ive의 어원 변천사

  1. Latin

    ‑īvus / ‑īva / ‑īvum

    완료 수동 분사에 붙어 "동작의 성질을 가진" 형용사를 형성. actīvus, creātīvus 등

  2. Anglo-Norman (~11–14c)

    ‑if (남성) / ‑ive (여성)

    actif, natif, sensitif 등의 형태로 중세 영어에 차용

  3. Middle English (~14c)

    ‑yf → ‑ive 전환

    네오라틴어의 영향으로 ‑if/‑yf 형태가 ‑ive로 재정립. 일부는 ‑y로 변화 (hastif → hasty)

  4. Modern English

    ‑ive 생산적 접미사로 정착

    active, creative, positive, negative 등 1,000개 이상의 형용사를 형성하는 핵심 접미사

Grammatical Function

접미사 ‑ive의 문법적 기능과 변형

‑ive 기본형 — act → active, pass → passive
‑ative 확장형 — creat(e) → creative, imagin(e) → imaginative
‑itive 확장형 — compet(e) → competitive, sens(e) → sensitive
‑sive 확장형 — explo(de) → explosive, expan(d) → expansive

주요 패턴: 동사 어간 + ‑ive → 형용사. 라틴어에서 완료 수동 분사(perfect passive participle)에 ‑īvus를 붙이는 패턴을 그대로 계승했습니다. 영어에서는 명사에 직접 붙기도 합니다 (sport → sportive).

Related Suffixes

‑ive와 함께 알아두면 좋은 관련 접미사

‑ivity ‑ive 형용사 → 명사 전환. creative → creativity, active → activity
‑ively ‑ive 형용사 → 부사 전환. creative → creatively, active → actively
‑iveness ‑ive 형용사 → 명사 전환 (대안형). effective → effectiveness
‑tion 같은 라틴어 동사 어간에서 명사를 형성. create → creation, act → action

FAQ

접미사 ‑ive에 관해 자주 묻는 질문

라틴어 ‑īvus에서 유래했습니다. 중세 영어 시기에는 앵글로노르만어 ‑if 형태로 차용되었고 (actif, natif 등), 14세기 이후 네오라틴어의 영향으로 ‑ive 형태로 재정립되어 현재에 이르렀습니다.

주로 형용사가 됩니다. "~하는 성질이 있는", "~하는 경향이 있는"이라는 뜻을 더합니다. 다만 detective(탐정), executive(임원), narrative(서사) 등 일부 단어는 명사로도 사용됩니다.

‑ative는 ‑ive의 확장형입니다. 라틴어 동사 어간에 포함된 연결모음 ‑at‑이 함께 나타난 형태로, creative(creat + ive), imaginative(imagin + ative) 등이 이에 해당합니다. 의미상 차이는 없으며, 어간의 형태에 따라 자연스럽게 결정됩니다.
